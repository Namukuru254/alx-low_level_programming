### Arguments to main

**argc** and **argv**

`argc
`

**argc** stands for `argument  count` which returns an `integer` number of arguments passed to the C program at runtime.

`argv`

**argv** stands for `argument vector`. A `vector` is a 1D array, thus `argv` is a 1D array of `strings`. Each of the `strings` is an `argument` passed to the program.
